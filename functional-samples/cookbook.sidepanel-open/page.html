Notes

This data contains active Commissioned Notaries Public from New York State Department of State (DOS). Each line will be either an individual or business licensee which holds business address and license number information. If the license type is an individual, the business name that the individual works for will be listed.

Last Updated: December 22, 2023

Data Provided By New York State Department of State (DOS)

Contact Info: licensing@dos.ny.gov

Commissioned NYS Notaries Public Data Dictionary

Data Label Data Type Data Description
Commission Holder Name (e.g. ALLISON AARON) Text Displays the name of the individual commissioned (licensed).
Commission Number (UID) (e.g. 01AA6151925) Text For both traditional and electronic notary, it displays the commission (license) number.
Business Name (if available) (e.g. J & M SPECIAL EFFECTS) Text Displays the business name, if provided.
Business Address 1 (if available) (e.g. 524 SACKETT ST) Text Displays the business address, if provided.
Business Address 2 (if available) (e.g. *blank*) Text Displays a continuation of the business address, if provided.
Business City (if available) (e.g. BROOKLYN) Text Displays the city for the business address, if provided.
Business State (if available) (e.g. NY) Text Displays the state for the business address, if provided.
Business Zip (if available) (e.g. 11217) Text Displays the Zip code for the business address, if provided.
Commissioned County (e.g. Kings) Text Displays the County Name where the individual is commissioned (licensed).
Commission Type (Traditional or Electronic) (e.g. Traditional) Text Commission types are designated as being either a traditional notary or an electronic notary.
Term Issue Date (e.g. 08/28/2022) Date Displays the date the current commission (license) was issued.
Term Expiration Date (e.g. 08/28/2026) Date Displays the commission (license) expiration date.
Georeference (e.g. POINT(-73.987601639 40.679885107)) Point Open Data platform-generated geocoding information from supplied address components. Point-type location is the centroid of the address components provided and does not reflect a pecific address if the street address component is not provided. Point-type location is supplied in "POINT" (<geocoded longitude> <geocoded latitude>) format.
